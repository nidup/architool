# Architool

Experiments with Akeneo PIM code base, hexagonal architecture & DDD :rocket:

## Requirements

- Composer
- PHP 7.1
- Git
- Akeneo PIM dev version 1.8

## How To Install

Run `composr.phar update --prefer-dist` to install the tool's dependencies.
